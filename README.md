# 🍽️ FoodHub Data Analysis

## Project Overview

FoodHub is an online food delivery platform connecting customers with restaurants.

This project uses Exploratory Data Analysis (EDA) to understand customer ordering behavior, restaurant performance, delivery efficiency, and customer satisfaction. The insights can help FoodHub improve operations and enhance the customer experience.

---

## Business Objective

The goal of this analysis is to:

- Identify the most popular restaurants and cuisines.
- Analyze customer ordering patterns.
- Evaluate food preparation and delivery times.
- Understand customer ratings and satisfaction.
- Generate business insights to improve FoodHub's services.

---

## Dataset

The dataset contains information about food orders placed through FoodHub, including:

- Order ID
- Customer ID
- Restaurant Name
- Cuisine Type
- Order Cost
- Day of the Week
- Customer Rating
- Food Preparation Time
- Delivery Time
